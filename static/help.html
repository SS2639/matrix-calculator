スカラー（数値）：行列セル以外にフォーカスがあるとき、数字・小数点・指数表記（e/E と指数部の +/- のみ、例 1e-6）をキーボードで入力すると式バーに数値ブロックができます（演算子や括弧の前後で確定します）。数値を入力しているあいだは Enter でその数値をトークンとして確定できます。式全体の計算は = です。+ - * / ** は数値に含めず、いずれも演算子トークンとして扱います（負の数は - トークンのあとに数値トークン）。val ボタンは、キーボード入力と同じ数値下書き（破線枠）を空の状態から開きます。空のときに打てる先頭文字は 0–9 と . のみです。空のまま Backspace、または先頭に使えないキーを押すと閉じます。一度入力したあと中身を空に戻しても閉じます。

演算子（キー）：+ - * / はキーボード・テンキー（NumPad）から入力できます。* を続けて押すと **（累乗）にまとまります。

変数記号（キー）：許可された1文字記号は1回のキーで1トークンとして式に入ります（連続入力による複数文字の変数名はありません）。大文字は小文字に揃えます。開発者向けのキー割り当ての全体は docs/KEYBOARD.md を参照してください。

n乗：指数としての記号 n（整数パラメータ）を表します。キーの n で式に n が入り、例としてスカラー 3 のあとに押すと 3**n になります。行列トークンのあとでも同様に __M…**n となります。
